The right-hand side of the vector ODE, a callback from CVODE.

Classes implementing this function should name it rhsFunc().

This function matches the prototype of CVRhsFn:

typedef int (*CVRhsFn) (realtype t, N_Vector y, N_Vector ydot, void *f_data)

This is the function in the expression as required by CVODE. Both and are vectors.

CVODE requires that this be a static function, but the last parameter allows this to act like a regular member function. The "myself" pointer is treated as the "this" pointer is within a regular member function.

Parameters
tThe current value of the independent variable, time in this case.
zThe input vector, as generated by initial conditions or by CVODE
z_primeThe output vector, to be filled by this function.
myselfA pointer to an AbstractProblem object where the function should get its configuration values.
