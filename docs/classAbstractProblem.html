Detailed Description

Abstract parent class for a CVODE problem.

All CVODE problems should be implemented as a child class of this class. Although all methods are virtual and thus can be overridden, those functions which are more commonly reimplemented in child classes are grouped at the top of this file.

The most important methods are rhsFuncAbstract(), denseJacAbstract(), and recordData() and friends. Child classes will typically also add some methods to set parameters and options.
