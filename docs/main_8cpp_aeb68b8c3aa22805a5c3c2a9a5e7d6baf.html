Split a string into a vector of realtype values.

In the following example, the vector v is filled with the four values 1.0, 2.7, 3.2, and 4.03:

vector<realtype> v;
v = split_realtype(",", "1.0,2.7,3.2,4.03");

The implementation does not try to be particularly efficient. It does, however, do a pretty good job of being correct.

Parameters
sepDelimeter on which to split. sep is stripped from the output.
textText to split.
