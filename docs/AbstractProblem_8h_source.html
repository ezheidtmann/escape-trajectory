33  /**
34  * The right-hand side of the vector ODE, a callback from CVODE.
35  * Classes implementing this function should name it \em rhsFunc().
36  *
37  * This function matches the prototype of \c CVRhsFn:
38  *
39  * <tt>typedef int (*CVRhsFn) (realtype t, N_Vector y, N_Vector ydot,
40  * void *f_data)</tt>
41  *
42  * This is the function \f$f(t,z)\f$ in the expression
43  * \f$z^\prime = f(t,z)\f$ as required by CVODE. Both \f$z^\prime\f$
44  * and \f$z\f$ are vectors.
45  *
46  * CVODE requires that this be a static function, but the last parameter
47  * allows this to act like a regular member function. The "myself"
48  * pointer is treated as the "this" pointer is within a regular member
49  * function.
50  *
51  * @param t The current value of the independent variable, time in this
52  * case.
53  * @param z The input vector, as generated by initial conditions or
54  * by CVODE

56  * @param myself A pointer to an AbstractProblem object where the function
