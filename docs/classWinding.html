Detailed Description

Utility class to count winding number for the RestrictedThree problem.

The winding number is the number of radians through which the projectile has traveled around a certain center of rotation: If the projectile takes a loopy path, this number should be large. If it makes a straight shot out of the system or performs a number of switchbacks, this number should be small.
